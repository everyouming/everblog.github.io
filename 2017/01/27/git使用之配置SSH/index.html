2017-01-27 • 分类: 技术 • 标签: git

git是分布式的代码管理工具，远程的代码管理是基于ssh的，所以要使用远程的git则需要ssh的配置。

三、配置SSH

  1. 登录到github
  2. 在个人中心的Settings里面找到SSH and GPG keys
  3. 点击右上角New SSH key
  4. 这是会要求填写Title和Key,Title填你注册github时的邮箱,Key填写之前生成的id_rsa.pub里面的内容即可
